# MindBalance

MindBalance es un prototipo funcional de una PWA orientada al bienestar emocional y al seguimiento de micro-hábitos en estudiantes universitarios. Esta entrega toma como base la formulación del proyecto en `Documentacion/` y materializa el MVP en una interfaz navegable construida con React, Vite y Chart.js.

## Qué incluye esta versión

- Registro, inicio de sesión y recuperación de contraseña simulada
- Dashboard con check-in emocional diario
- Recomendaciones personalizadas según el estado emocional
- Alertas preventivas por patrones de riesgo
- Gestión completa de micro-hábitos
- Historial emocional y de hábitos con edición/eliminación de registros
- Configuración de perfil y recordatorios
- Acceso a canales de bienestar estudiantil
- Persistencia local con `localStorage`
- Manifest y service worker básicos para comportamiento tipo PWA

## Ejecutar el prototipo

```bash
cd frontend
npm install
npm run dev
```

Build de verificación:

```bash
cd frontend
npm run build
```
